🏥 HealthMate – Smart Healthcare Management System
🚀 Overview

HealthMate is a full-stack healthcare management web application designed to help users monitor, analyze, and manage both physical and mental health in one platform.

It provides real-time health insights, secure data handling, and downloadable reports, making it a practical and user-friendly healthcare solution.

✨ Features
👤 User Features
🔐 Secure Signup & Login (Password Hashing)
📊 Personal Dashboard with Health History
🩺 Add & Track Health Records (BMI, BP, Sugar, etc.)
🧠 Mental Health Checkup System (Score + Summary)
📄 Download Health Reports in PDF format
🏥 Nearby Doctors Information
🚪 Logout Functionality
🛠️ Admin Features
👥 View All Users
📋 Access User Health Records
✏️ Edit / Delete Records
📥 Download Reports
🔐 Role-Based Access Control
🧠 Health Analysis Logic
BMI Calculation & Classification
Blood Pressure Monitoring
Heart Rate & Sugar Level Analysis
Personalized Health Tips Generation
Mental Health Scoring System

⚙️ Installation & Setup
# Clone the repository
git clone https://github.com/princesharmabackend/healthmate.git

# Navigate to project folder
cd healthmate

# Install dependencies
pip install -r requirements.txt

# Run the application
python app.py
